lazyGet( \Aura\Di\string $key ) : \Aura\Di\Lazy

Returns a Lazy that gets a service. This allows you to replace the following idiom ...

$di->params['ClassName']['param_name'] = new Lazy(function() use ($di)) { return $di->get('service'); }

... with the following:

 $di->params['ClassName']['param_name'] = $di->lazyGet('service');

Returns a Lazy that creates a new instance. This allows you to replace the following idiom:
