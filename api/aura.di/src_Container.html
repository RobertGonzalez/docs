lazyGet( string   $key, ) : aura\di\Lazy

Description

Returns a Lazy that gets a service. This allows you to replace the
following idiom ...
$di->params['ClassName']['param_name'] = new Lazy(function() use ($di)) {
return $di->get('service');
}

... with the following:

$di->params['ClassName']['param_name'] = $di->lazyGet('service');

Arguments

Name Type Description Default
$key string The service name; it does not need to exist yet.

Return value

Type Description
aura\di\Lazy A lazy-load object that gets the named service.

lazyNew( string   $class, array   $params = null, ) : aura\di\Lazy

Description

Returns a Lazy that creates a new instance. This allows you to replace
the following idiom:
$di->params['ClassName']['param_name'] = Lazy(function() use ($di)) {
return $di->newInstance('OtherClass', array(...));
}

... with the following:

$di->params['ClassName']['param_name'] = $di->lazyNew('OtherClass', array(...));

Arguments

Name Type Description Default
$class string The type of class of instantiate.
$params array Override parameters for the instance. null

Return value

Type Description
aura\di\Lazy A lazy-load object that creates the new instance.
