Description

Text formatting for VT100 terminals.

\array $format = 'array'

Array of format conversions for use on a variety of pre-set console style combinations.

Based on ANSI VT100 Color/Style Codes according to the VT100 User Guide and the ANSI/VT100 Terminal Control reference. Inspired by PEAR Console_Color.
