exec( ) : \void

Executes the Page action. In order, it does these things:

  • signals 'pre_exec'

  • is the action is not to be skipped ...

    • signals 'pre_action'

    • calls invokeMethod() to run the action

    • signals 'post_action'

  • signals 'post_exec'
