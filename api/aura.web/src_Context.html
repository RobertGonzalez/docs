getInput( string   $key = null, string   $alt = null, ) : mixed

Description

Retrieves an **unfiltered** value from a user input.
A value by key from the `$post` *and* `$files` properties, or an
alternate default value if that key does not exist in either location.
Files takes precedence over post.

If the key is null and the content type isn't `multipart/form-data` and
`$post` and `$files` are empty, the raw data from the request body
is returned.

Arguments

Name Type Description Default
$key string The $post and $files key to retrieve the value of. null
$alt string The value to return if the key does not exist in either $post or $files. null

Return value

Type Description
mixed The value of $post[$key] combined with $files[$key], or the raw request body, or the alternate default value.
