headerLabel( string | $label, ) : string

Description

Sanitizes header labels by removing all characters besides [a-zA-z0-9_-].

Underscores are converted to dashes, and word case is normalized.

Converts "foo \r bar_ baz-dib \n 9" to "Foobar-Baz-Dib9".


Arguments

Name Type Description Default
$label string | The header label to sanitize.

Return value

Type Description
string The sanitized header label.

headerValue( string | $label, string | $value, string | $charset = utf-8, ) : string

Description

Sanitizes a header value, then encodes and wraps as per RFC 2047.

Copied, with modifications, from the "mime.php" class in the PEAR Mail_Mime package (v 1.62). Takes the added step of "un-wrapping" the value (newline-and-space) and then removing all control characters (including newlines) before encoding and re-wrapping.


Arguments

Name Type Description Default
$label string | The *sanitized* header label; needed for line length calculations.
$value string | The header value to encode.
$charset string | The character set to note when encoding. utf-8

Return value

Type Description
string The encoded header value.
