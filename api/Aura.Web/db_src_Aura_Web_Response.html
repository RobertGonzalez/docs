Retains information about the controller results for transfer to an actual HTTP response object.

Properties

bool  $cache= 'null'

Should the response disable browser caching?

When false, the getHeaders() method will set these headers in its return value (note that they are not added to $headers directly):

{{code: Pragma: no-cache Cache-Control: no-store, no-cache, must-revalidate Cache-Control: post-check=0, pre-check=0 Expires: 1 }}

When true or null, the getHeaders() method will make no changes to the existing headers.
