  protected  $format

Array of format conversions for use on a variety of pre-set console style combinations.
<p>Based on <code>ANSI VT100 Color/Style Codes</code> according to the <a href="http://vt100.net/docs/vt100-ug">VT100 User Guide</a> and the <a href="http://www.termsys.demon.co.uk/vtansi.htm">ANSI/VT100 Terminal Control reference</a>. Inspired by <a href="http://pear.php.net/Console_Color">PEAR Console_Color</a>.</p>
