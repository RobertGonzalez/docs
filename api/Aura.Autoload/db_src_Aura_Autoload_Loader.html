Parameters
Name Type Description
$name string

The class name prefix, e.g. 'Aura\Framework\' or 'Zend_'.
$paths array | string

The absolute path leading to the classes for that prefix, e.g. '/path/to/system/package/Aura.Framework-dev/src'. Note that the classes must thereafter be in subdirectories of their own, e.g. `'/Aura/Framework/'.
addPrefixes( array $prefixes = array() ) : void

Add an array of prefixed name spaces

An array of associative name and paths.

paths can also be an array or a string

Eg :

$loader->addPrefixes(array( 'Zend_'=> '/path/to/zend/library', 'Aura' => array( '/path/to/project/Aura.Router/src/', '/path/to/project/Aura.Di/src/' ), 'Vendor' => array( '/path/to/project/Vendor.Package/src/', ), 'Symfony/Component' => 'path/to/Symfony/Component', ));
