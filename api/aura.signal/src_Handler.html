exec( object   $origin, string   $signal, array   $args, ) : null|array

Description

Execute the handler: if the originating object matches the required
sender, and the signal matches the required signal, then run the
callback and return the results.

Arguments

Name Type Description Default
$origin object The originating object that sent the signal.
$signal string The signal sent by the originating object.
$args array Arguments for the callback, which will be invoked if the sender and signal match this Handler.

Return value

Type Description
null|array An array of parameters suitable for creating a Result object, or null if the origin and signal did not match this Handler.

object|string  protected  $sender

Handle signals from this sender. This can be an object, in which case
the Handler will match only signals from that object, or a class name,
in which case the Handler will match signals from any instance of that
class.
