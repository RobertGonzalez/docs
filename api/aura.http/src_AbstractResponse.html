setHeader( string | $name, string | $val, bool | $replace = true, ) : aura\http\Response

Description

Sets a header value in $this->headers.

This method will not set 'HTTP' headers for response status codes; use the [[aura\http\Response::setStatusCode() | ]] and [[aura\http\Response::setStatusText() | ]] methods instead.


Arguments

Name Type Description Default
$name string | The header label, such as "Content-Type".
$val string | The value for the header.
$replace bool | This header value should replace any previous values of the same key. When false, the same header key is sent multiple times with the different values. true

Return value

Type Description
aura\http\Response This response object.
